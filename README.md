# AutomacaoGOV

Projeto de automação de testes desenvolvido com Cypress como trabalho final da disciplina.

## 🚀 Objetivo
Automatizar fluxos de uma aplicação web simulando o comportamento do usuário e validando funcionalidades.

## 🧪 O que foi testado
- Fluxo de login
- Navegação entre páginas
- Validação de campos
- Cenários positivos e negativos

## 🛠️ Tecnologias
- Cypress
- JavaScript
